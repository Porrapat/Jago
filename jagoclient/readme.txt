Copyright R. Grothmann & Askwyhale

Jago is a client for the internet go servers,
and an editor for the SGF go format. 

This version will install Jago on a Windows
computer. It wills setup several icons for
various Java versions. At least one must
be installed.

Additionally, it will create an icon for 
the HTML-documentation for Jago.

Jago homepage at
