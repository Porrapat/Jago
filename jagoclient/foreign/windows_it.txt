Utilizzare le finestreMolti dei dettagli che seguono non sono applicabili a tutti i sistemioperativi.Jago memorizza la posizione e la dimensione delle finestre nel fileGO.CFG.  Il programma cerca di scrivere il file nella cartellaattiva. E' necessario quindi che tu abbia accesso in scrittura allacartella attiva. Nei sistemi multiutente devi creare per ciascunutente una cartella locale per Jago utilizzando uno scriptd'installazione. Finita l'installazione lo script fara' partire Jagodalla cartella locale. Devi passare la cartella di Jago come argomentocon il parametro -home.Molti sistemi operativi permettono l'utilizzo della funzione copia eincolla. In Windows per copiare il testo premi i tasti CTRL-C. Perincollare nel campo di input di Jago premi i tasti CTRL-V. Inalternativa puoi premere il tasto destro del mouse che fara' apparireun menu specifico. A meno che tu non abbia selezionato l'opzione"Usare AWT TextArea" (nelle opzioni avanzate) Jago utilizza dellefinestre dedicate per visualizzare i messaggi. Queste finestrepermettono l'utilizzo di testo colorato e non hanno limitazioni nelledimensioni. E' possibile copiare il contenuto negli appunti.Le finestre di dialogo possono essere chiuse utilizzando il tasto ESC(solo quando sono in primo piano).I comandi inviati al server utilizzando l'apposito campo di inputvengono memorizzati. Per richiamare i comandi precedentemente inviatiutilizza il tasto freccia (su o giu').
