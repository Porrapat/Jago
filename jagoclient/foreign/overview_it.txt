IntroduzioneSe hai gia' giocato a Go utilizzando i Go server presenti su Internetnon avrai alcun problema con questo programma: basta definire iparametri di connessione e connettersi. Per farlo puoi utilizzare unaprocedura automatica oppure inserire manualmente nome e password.Se stai scoprendo ora il mondo del Go su Internet leggi ladocumentazione in formato HTML (in inglese) fornita con questoprogramma o consulta i vari file di aiuto.Con Jago puoi anche giocare a Go direttamente con un amico suInternet, senza bisogno di collegarsi ai Go server,col vantaggio di un calcolo piu' accurato del tempo. Permaggiori informazioni vedi il file di aiuto sulle "Partite conpartner".  Questa caratteristica puo' essere utilizzata per creareuna piccola rete di server aperti.Jago e' anche un editor SGF per partite di Go. Puo' creare variantidi gioco, commenti, nomi per i nodi, lettere, segni e molto altroancora. Esiste anche una funzione per la ricerca di testo neicommenti.
