CodificaQuesta finestra permette di impostare la codifica relativa allascrittura e alla lettura dei file locali. Se non sei sicuro del tuocodepage locale cancella la stringa e premi OK. Jago impostera'automaticamente il codepage di default.Questa opzione serve per leggere i file SGF in lingue straniere.  Adesempio Cp850 permette a Jago di interpretare correttamente i filescritti in DOS e 8859_1 e' il codepage di default per la linguaitaliana.  Ovviamente, se utilizzi le versioni occidentali dei sistemioperativi, non puoi aspettarti di visualizzare i commenti scritti coicaratteri giapponesi a meno che tu non abbia installato il set dicaratteri appropiato.  Un altro codepage molto utile e' UTF-8 chescrive in formato UTF-8 codificato Unicode. E' possibile utilizzareanche lo stesso Unicode ma in questo caso ben pochi riusciranno aleggere i file SGF che realizzerai.Puoi impostare il codepage per ognuna delle connessioni. Per farloutilizza il pulsante "Imposta" nella finestra "Connessioni server" einserisci il codepage nel campo "Codifica".
