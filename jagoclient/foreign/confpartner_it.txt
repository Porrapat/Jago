Configurazione delle Connessioni con PartnerLe "Connessioni con Partner" sono connessioni instaurabilidirettamente con degli amici utilizzando la rete senza bisogno diconnettersi ai server dedicati come IGS.  Viene aperta una finestrachat e, quando decidi di iniziare una partita, una finestra con latavola da gioco. Diversamente dalle connessioni con gli Internet GoServer nelle connessioni con partner il conteggio del tempo e' moltoaccurato.Entrambi i partner devono avviare un server ed attendere laconnessione. Per instaurare la connessione uno dei partner devepremere il pulsante "Connetti". Fatto questo la connessione hainizio. E' possibile configurare la porta del server ma se decidi dimodificare il valore di default 6970 devi comunicarlo in anticipo altuo partner. Inoltre tieni presente che devi assegnare un solo nome altuo server.Per configurare la tua connessione partner premi il pulsante "Imposta"o "Aggiungi". Devi inserire i seguenti dati:- l'indirizzo internet del tuo partner- il numero porta del tuo partner (utilizza 6970)- lo stato del partner (vedi sotto)Per configurare il tuo server seleziona la voce "Il tuo nome" nel menu"Partner" ed inserisci il dato richiesto. Il formato suggerito e' "TuoNome (Grado)".Il tuo partner deve avviare il server selezionando la voce "AvviaServer" nel menu "Server". Il server avviato attende la connessionenella porta specificata. Per connetterti premi il pulsante"Connetti". Buona partita!Se avvii un server pubblico Jago lo comunichera' a tutti partnerelencati nella lista connessioni partner a cui non sia stata assegnatalo stato "Silente". A loro volta i server contattati provvederanno acomunicare l'avvio del tuo server a tutti i server loro connessi a cuinon sia stato assegnato lo stato "Privato". Nel caso sia assegnato lostato "Locale" il server in questione comunichera' l'avvio ai serverconnessi ma questi non inoltreranno la comunicazione ad altri server.Inoltre, ogni server che riceve la comunicazione dell'avvio del tuoserver ti spedira' una lista dei server aperti di cui e' a conoscenza(rispettando le condizioni di stato sopraddette). Puoi vedere la listadei server aperti premendo il pulsante "Server aperti?".Utilizzando questa caratteristica di Jago e' possibile creare una retedi server.Le comunicazioni descritte vengono effettuate spedendo dei datagrammiai partner. A questo scopo viene utilizzata una porta. Questa portacorrisponde a quella del partner piu' 2 (normalmente 6972). Allo stop del tuo server i tuoi partner riceveranno il messaggio"Server is busy". Comunque, la thread, che tiene nota dei serveraperti continua a girare.
