Questo vi consente di impostare un relay server. Dovrete usarlosoltanto se utilizzate Jago per mezzo di un browser. In questo caso ilvostro server (quello che contiene l'applet Jago) fa partire un relaydemon. Questo viene di solito preimpostato dall'amministratore delserver.In ogni caso si puo' evitare di usare un relay server riducendo ilivelli di sicurezza per gli applet Java (Questa operazione e'consentita in Internet Explorer).
