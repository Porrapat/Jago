Настройка партнерских подключений

Партнерские подключения - это подключения к друзьям через Интернет. В
вашем распоряжении окно для разговора (чат) и, при начале игры, окно
доски. В отличие от го-серверов Интернета, здесь расчет времени точный.

Любой из партнеров должен запустить сервер, ожидающий подключений. Другой
партнер для соединения с этим сервером нажимает кнопку "Подключить". Эта
процедура устанавливает соединение. Вы можете изменить порт сервера, но
лучше оставить 6970. Кроме этого, вы должны назначить уникальное имя
вашему серверу.

Для настройки партнерского соединения нажмите на панели управления
кнопку "Редактировать" или "Добавить". Вам придется ввести:

- Адрес Интернет вашего партнера.
- Порт вашего партнера (должен быть 6970)
- Статус партнера (см. ниже)

Для того, чтобы сконфигурировать ваш сервер, установите ваше имя с
помощью соответствующей команды меню, известив о себе своего партнера.
Предлагаю использовать формат "Ваше Имя (уровень)". Не изменяйте порт
с 6970, установленного по умолчанию. Хотя вы можете это сделать, нужно
заручиться согласием ваших партнеров, собирающихся подключаться к вам.

Ваш партнер должен запустить сервер соответствующей командой меню.
Стартовавший сервер ожидает запроса с указанного порта. Для соединения
нажмите кнопку "Подключить" в нижней части окна. Всё - наслаждайтесь
общением и игрой!

Если вы создаёте общедоступный сервер, Jago разошлет это по всем
партнерским подключениям, кроме находящихся в состоянии "молчащий" (silent).
Эти серверы сообщат вашему о запуске других подключенных серверов, 
если только те не в состоянии "частный" (private). Установка
"местный" (local) означает, что партнеры могут оповещать подключенные
сервера о запуске вашего сервера, но те не могут рассылать это дальше.
Это определяется отметкой соответствующего состояния в меню. 

Кроме этого, любой сервер, получивший сообщение о запуске вашего сервера, 
пошлет назад список известных ему открытых серверов, подчиняясь правилам
конфиденциальности, описанным выше. Вы можете просмотреть список открытых
серверов с помощью кнопки "Открытые?".

Используя эту возможность вы можете строить большую или меньшую сеть
открытых серверов.

Описанная связь осуществляется посылкой дейтаграмм партнерам. Для этого
используется порт. Номер этого порта - порт партнера плюс 2 - обычно
6972.

Остановка вашего сервера вызовет сообщение "Сервер занят" ("server is busy"), 
предупреждающее ваших партнеров. Однако поток, принимающий информацию 
об открытых серверах будет работать и далее.
