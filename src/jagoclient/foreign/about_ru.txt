JagoClient, Russian version
Клиент Го-сервера Интернета (Internet Go Server) и программа просмотра SGF,
программирование и авторские права принадлежат Рене Гросманну (Rene Grothmann).

Эта программа бесплатна в соответствии с Gnu General Public License.

Автор не несет ответственности за работоспособность данной программы.
Вы используете её исключительно на свой риск.

За новыми версиями обращайтесь по адресу:

"http://www.rene-grothmann.de/jago/".

В случае проблем или ошибок, посылайте e-mail по адресу
"jago@rene-grothmann.de".

Изменения после версии 4.00:

4.00
- Новый вид окон доски. Размещение значков для удобного редактирования.
4.01
- Для подключенных досок Го добавлен значок посылки сообщения.
- Значки редактора доски удалены с подключенных досок Го.
- Исправлено размещение варианта на втором ходе дерева игры.
- Добавлены тексты справки для значков.
4.02
- Попытка решения некоторых редких Telnet проблем с IGS. Похоже удачная.
4.03
- Исправлено странное поведение после команды "Показывать кнопки".
4.04
- Некоторое улучшение значков.
- Из списка шрифтов теперь можно выбрать шрифты Courier, TimesRoman и Helvetica.
4.05
- Новые инсталляторы теперь используют InstallAnywhere!
4.06
- Добавлены китайские ресурсы в InstallAnywhere.
- Поддерживается минимальный Windows инсталлятор для Java 1.3.
4.07
- Решена проблема звука. Теперь звук работает с Java 1.1, используя sun.audio,
  и с Java 1.3, используя новый Media API.
- Поле Отправить в окне доски теперь отправляет пустые линии.
- Диалог Отправить теперь в центре доски.
- Редактор шрифтов теперь запоминает свою предыдущую позицию.
- Шрифт подключения теперь по умолчанию Monospaced.
4.08
- Улучшены некоторые звуковые события.
4.09
- Новые возможности для вариантов. Они предназначены для просмотра файлов
  созданных MGT или другими программами, которые не показывают на доске
  наличие вариантов.
- Теперь можно с помощью мышки создавать варианты на доске наблюдения.
- Заменены некоторые системные цвета фона на цвета, задаваемые пользователем.
4.10
- Несущественные косметические изменения.
- Инсталлятор Windows теперь проводит установку для нескольких версий Java.
4.11
- Убрана ошибка справки в диалоге, проявившаяся в Java 1.3.
- Удаление игр теперь работает только для более чем двух игр.
4.12
- Обновлены запись и чтение XML формата.
- Обновление панели значков (особенно инструмента подсказок).
4.13
- Jago можно запустить командой "java -jar jago.jar" в Java 1.3.
- Исправлен JagoGame апплет.
4.14
- Отмена теперь работает в GMP играх.
4.15
- SGF больше не записывает теги с пустыми действиями (такие как KM[]),
  (благодарю Эда Гаилларда (Ed Gaillard) за это указание).
4.16
- Небольшое исправление поведения "Удалить камень" (благодарю
  Гарольда Боегехольца (Harald Boegeholz)).
  4.17

- Решена проблема с нумерацией последнего хода.
4.18
- Новая версия инсталлятора Windows.
4.19
- Исправлено удаление белых камней (теперь удаляется AW тэг).
- Возможность продолжения нумерации в вариантах.
- Во фрейме названия теперь размещаются названия загруженных игр.
- Можно сохранять позицию.
4.20
- Фрейм названия и информация об игре теперь содержат имя файла
  сохраненных игр, если эти поля ранее были пустыми.
- Java Web Start.
- Файлы конфигурации сохранены в основной папке пользователя вместо
  текущей папки. Если необходимо, скопируйте старые файлы (*.cfg)

  Извините, это должно было быть сделано!
4.21
- Исправлена ошибка, возникающая при закрытии IGS подключений
  закрытием  окна в Linux.
4.22

  когда игра сохранена.
- Кнопка ход вперед для посылки ">" в обучающих играх.
4.23
- Если не используется параметр -h, конфигурационные файлы всегда в
  основной папке пользователя.

4.24

4.25
