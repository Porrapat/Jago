Использование Windows

Многое зависит от вашей операционной системы.

На Windows 95 с немецкой клавиатурой, вы не можете непосредственно ввести
знак @, например для регистрации. Это ошибка Java. Вместо этого нажмите
клавишу ALT, наберите 64 и отпустите.

JAGO сохраняет размер и положение своих окон в файле GO.CFG. JAGO записывает 
этот файл в активном каталоге. Поэтому, вам нужны права на запись в этот 
каталог. При использовании многопользовательских сред вы должны установить 
местный каталог JAGO для каждого из пользователей с установочным скриптом. 
Затем этот скрипт может запускать JAGO из этого каталога. Вам нужно указать 
путь к каталогу JAGO в качестве дополнительного аргумента с параметром -home, 
чтобы получить доступ к текстам справки(см. "Справочная система").

Большинство систем позволяют вырезать и вставлять информацию из текстовых
полей. В Windows нужно использовать CNTRL-C, чтобы скопировать текст
откуда-либо, и CONTRL-V чтобы вставить в поле ввода JAGO. Либо используйте
правую кнопку мыши для доступа контекстного меню для копирования. JAGO
использует замещение для текстовых областей, если только вы не выбрали
обычные текстовые области в меню "Дополнительные настройки". Эта опция разрешает
цвет в выводе и не имеет ограничений по размеру текста. Замещение может быть 
скопировано в буфер обмена. 

Диалоговые окна можно закрывать кнопкой Escape, если они обладают фокусом.

Текстовые поля ввода, посылающие команды серверу, имеют историю. Вы можете
вызвать старые команды клавишей "Вверх".
